## Math Formulas

Formulas render as pretty-print CAS expressions.

```math
integrate(sin(x)^2,x)
```

```math
sum(1/n^2,n,1,infinity)
```

```math
(x^2+1)/(sqrt(x)+3)
```

```math
limit(sin(x)/x,x,0)
```

```math
diff(x^3+2*x,x)
```
